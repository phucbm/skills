# Meta

Skills for managing this repo itself.

## Model-invoked

- **[add-skill](./add-skill/SKILL.md)** — Add a new skill or knowledge entry to phucbm/skills repo.
- **[update-skill](./update-skill/SKILL.md)** — Diff and update an existing skill entry.
- **[learn](./learn/SKILL.md)** — Scan any repo, diff against existing skills, and suggest what to save or update.
- **[claskit](./claskit/SKILL.md)** — Autonomous Claude Code task runner — write task specs as Markdown, run claskit, watch Claude implement them.
